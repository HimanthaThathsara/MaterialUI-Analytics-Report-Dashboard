##  Overview

<code>A Material-UI (React + MUI) analytics/report dashboard with many pre-built chart components and a theme system. 
The repo includes a large set of chart components (ApexChart, React Google Charts, React Gauge, Circular Progressbar, Liquid Gauge, MUI-based charts) and theme/shared-theme code.</code>
